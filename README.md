# MacroSpend Tracker

MacroSpend Tracker is a lightweight desktop agent that provides real-time visibility into how software is actually used across an organization.

It runs quietly in the background and streams usage data to the MacroSpend dashboard, enabling companies to identify unused licenses, reduce spend, and make data-driven decisions.

---

## What it does

- Tracks active application usage (app name, duration)
- Sends data securely to the MacroSpend platform
- Runs as a background tray application
- Supports macOS and Windows

---

## What it does NOT do

- Does not record keystrokes
- Does not capture screenshots
- Does not access files or personal content

---

## Installation (Beta)

1. Download the latest release:
   https://github.com/macrospend/macrospend-tracker/releases/latest

2. Install the application
3. Open the MacroSpend dashboard
4. Register your device and launch via:
   `macrospend://register?key=...`

The tracker will begin syncing automatically.
